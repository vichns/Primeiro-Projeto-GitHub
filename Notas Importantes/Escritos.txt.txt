|git status|

|git add .|  ou  |git add -A|                                               (Adicionar todos os arquivos)

|git commit -m "Inclusão de anotações ao GitHub"|                           (Inclusão de anotações ao GitHub)

|git config --global user.name "vichns"|                                    (Usuário da conta " ")

|git clone " "|                                                             (Colar o endereço code https da sua conta GitHub na nuvem)

|git push origin main|                                                      (Enviar todos os arquivos para a nuvem github)
